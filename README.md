 # Digital Voting Machine (Verilog)

## Description
This project implements a Digital Voting Machine using Verilog HDL.  
The system allows voters to select a candidate and records the vote digitally.

## Tools Used
Vivado  
ModelSim / QuestaSim

## Files
voting_machine.v – Verilog design file  
voting_machine_tb.v – Testbench for simulation
