 🗳️ Digital Voting Machine (Verilog)
📌 Description

This project implements a Digital Voting Machine (DVM) using Verilog HDL.
The system enables voters to securely cast their votes by selecting a candidate through input switches. Each vote is counted and stored digitally, ensuring accuracy and reliability.

The design includes:

Multiple candidate selection

Vote counting mechanism

Reset functionality

Result display through output registers

⚙️ Tools Used

🛠️ Vivado – For synthesis and implementation

💻 ModelSim / QuestaSim – For simulation and verification

📂 Files Included

📄 voting_machine.v – Main Verilog design file

🧪 voting_machine_tb.v – Testbench for simulation

🔣 System Symbols / Representation
Symbol	Meaning
🗳️	Voting Machine
👆	Voter Input
🔘	Candidate Selection Button
🔢	Vote Counter
🔄	Reset
📊	Result Display
🧠 Working Principle

👆 Voter presses a button corresponding to a candidate

🔘 Input is detected by the system

🔢 Vote count for that candidate increments

🔄 Reset clears all votes when required

📊 Results are displayed digitally
